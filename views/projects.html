Wordle Game

A group and I collaborated to create a version of NYTimes Wordle in Python. We made this game on a shared Google Colab document. Other than standard game play the program also had the ability for users to input their own word to guess, ability for users to choose the length of the random word, and ability to see their overall game statistics.

Message Encrpytion Program

I developed an encryption program in C++. The program allowed a user to choose between three encryption methods; the standard Caesar Cipher, the Vigenere Cipher, and a sequence of the two. A user could also decrypt their messages using the reverse methods. This program had several functions for each method.

Odyssey Game

I developed a two-player online boardgame based on the Odyssey. Players would take turns rolling a dice to move across the game board and land on multi-colored tiles. Depending on the color of the tile the player would either face a calamity, combat, or an anagram to descramble. I utilized Object-Oriented Programming and incuded three classes and two structs.

File System Simulation

A partner and I created a Linux file system simulation using C++. We utlized binary search trees and graphs for efficiency. Additionally, we made our own GitHub repository in which we created each command's functionality in different branches and followed version control best practices to combine our work.

Planar Graph Poster

A group and I researched the essential properties of Planar Graphs and how those properties allow us to define them in a new way. We created a poster that was presented using LaTeX and Google Drawing for our homemade images and diagrams.
